## Running Locally

Clone the project

```bash
https://github.com/TheDanniCraft/Portfolio.git
```

Go to the project directory

```bash
cd Portfolio
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
